Dynamical Systems in Neuroscience : Chapter 1

13 Sep, 2016 · Read in about 3 min · (499 Words)
dynamical systems neuroscience theory

Intersting things in the first reading

Introduction

  • HH Model is not an integrator, but a resonator.
  • The term thresold doesn’t make much sense.:
    • There is no one precise voltage level
    • Rheobase doesn’t make much more sense either, because sometimes cells will fire for the same current after much longer. It also can result in change in spike height to a point where it is difficult so say whether something is a spike or not.
    • Also, neurons can fire to inhibitory inputs. (rebound spikes)
    • Neurons show resonance, so certain types of inputs get spikes, but faster or slower inputs don’t.
  • Getting paramters of neurons is hard. And often makes no sense.

Dynamical systems

Classify neuronal state variables:

  • membrane potential ( $V_m$ )
  • excitation variables ( ex. Na activation current)
  • recovery variables (ex. Na inactivation current)
  • adaptation variables (ex .Ca dependent current)

Bifurcations

There are fundamentally just 4 types of bifurcations (where codimention =1 i.e. only one variable is changed to switch behaviour) (I wonder how I can prove this.)

  • saddle-node bifurcation : a node and a saddle combine to disappear. they are outside a nearby limit cycle (or another stable equilibrium, I assume)
  • saddle node on invariant circle bifurcation : the saddle and the node are on the limit cycle like invariant cicle. They combine and then the circle becomes a limit cycle.
  • subcritical andronov-hopf bifurcation : there is an unstable limit cycle between a node and a stable limit cycle. the unstable one shrinks to the node to combine and become an unstable node.
  • supercritical andronov-hopf bifurcation : a stable point blows up into a limit cycle with an unstable point at the centre

The andronov-hopf bifurcations have 2 stable states coexisting. So, the system can be made to switch between them by a brief input. Also, this system shows hysterisis for appropriate inputs.

The book does some simple analysis of these at this point, but is very handwaving. I hope it will be covered better in later chapters. But this did lead me to understand to some extent why definig thresold is hard and what its true nature is like. (it is not a single voltage, but a line (surface) in the phase space.)

About building models

He makes a good point about how to make good models of neurons. He says that it is not enought to get the right currents or even get the correct rheobase and input resistance and firing frequency. What is important is that it has to have the corect neurocomputational features. List:

  • co-existance of resting and spiking states
  • spike latencies
  • subthreshold oscillations
  • rebound spikes, etc.

I like this list as I had not considered some of these while working on my project and should try to find relevant data sources for them.
